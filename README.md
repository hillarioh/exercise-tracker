    - Express
    - React.js

## How to get started with the project

Clone it to your local machine

### Frontend

cd into the root folder of the project
then run

`yarn start`

### Backend

change directory one level deep to backend folder

`cd backend/`

run nodemon server and you are good to go

`nodemon server`
